# Electronic Voting System using Blockchain and Smart Contracts

# Blockchain Voting System

By Ifra Shaaz Azeez

## Description
This project is a blockchain-based electronic voting system that ensures secure and transparent elections. It uses Ethereum smart contracts to store votes in a decentralized manner.

## Features
- Secure and tamper-proof voting
- One person one vote system
- Transparent vote counting
- Simple user interface

## System Overview
The system consists of:
- Web interface for voting
- Smart contracts for handling voting logic
- Basic authentication for users

## Smart Contracts Used
1. Voting (Ballot) Contract  
   - Stores candidate details  
   - Records votes securely on blockchain  

2. Election Contract  
   - Used to create and manage voting sessions  
   - Deploys ballot contracts  

## Modifications Made
- Updated user interface text  
- Added custom success message after voting  
- Improved project structure  
- Simplified documentation  

## Output
The system allows users to cast votes securely and displays the results transparently.

## Technologies Used
